WORKERS REGRESSION PROJECT

# INFO
This project use regression models like Linear Regression and Decission Tree for predict workers salary. Data base is made by myself using PostgreSQL.

# TECHNOLOGIES
* Python 3.11
* Pandas 2.0.0
* Matplotlib 3.7.0
* Seaborn 0.12.2
* Sklearn 1.2.

# TABLE OF CONTENTS
1. Data analysis
2. Charts
3. Corrected data
4. Linear regression model
5. Finding best depth for decision tree model
6. Decission tree
